# Reference

Reference pages are for exact names, arguments, policies, and contracts.

Start with guides when you are learning a concept. Use this section when you
already know what you are looking for.

## Pages

- [Schema](schema.md): table classes, parse APIs, hooks, policies
- [Fields](fields.md): field declarations, IDs, discriminators, references
- [Parsers](parsers.md): parser factories and annotation inference
- [Errors](errors.md): `TableError`, `TableErrors`, and stable codes
- [Source And Context](source-context.md): `TableData`, `TableCell`, contexts
- [CellDSL](dsl.md): token, pattern, predicate, fallback, composition APIs
- [Transformers](transformers.md): table transformers and group expansion
- [CLI](cli.md): `talika check`, `talika describe`, checker APIs
- [Public API](public-api.md): top-level exports
